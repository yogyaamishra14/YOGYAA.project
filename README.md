# digital literacy project 
Digital Literacy Project showcasing practical skills in content creation, coding, professional networking, communication, and cyber safety.
Includes Canva design, LinkedIn/GitHub/Kaggle profiles, HackerRank problem, Google Form, emails, and a UPI fraud case study, promoting safe and effective use of digital tools.


🎯 Objectives
To develop a basic understanding of digital literacy and its importance
To learn how to use digital tools like Canva, Google Forms, and coding platforms effectively
To build a professional online presence using platforms like LinkedIn, GitHub, and Kaggle
To improve problem-solving skills through coding practice
To enhance professional communication skills by writing emails and guidelines
To understand responsible social media usage and digital behavior
To create awareness about cybercrime, especially UPI fraud, and ways to prevent it
To apply digital skills in real-world academic and practical scenarios.

Task 1 involved creating a visual design using a Canva-like tool to explain the concept of digital literacy in a simple and creative way.
Task 2 focused on setting up professional profiles on LinkedIn, GitHub, and Kaggle to build an online presence.
Task 3 included solving a coding problem on HackerRank and creating a Google Form, helping develop technical and content creation skills.
Task 4 required writing professional emails and preparing a checklist of do’s and don’ts for responsible digital usage, improving communication skills.
Task 5 was a case study on UPI fraud, which helped in understanding cyber threats and learning how to stay safe online.
